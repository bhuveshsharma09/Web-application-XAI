Explainable Artificial Intelligence

Platform to generate AI model explanation without coding

What is XAI?


According to Tech Target, Explainable AI (XAI) is artificial intelligence that is programmed to describe its purpose, rationale and decision-making process in a way that can be understood by the average person. XAI is often discussed in relation to deep learning and plays an important role in the FAT ML model (fairness, accountability and transparency in machine learning).

Features of Platform

No coding required

No coding required to explain data or AI mode. The user only has to upload Model and data files in .pkl format and select the preferred XAI method.

Model Explanation

Platform provides the user with local and global explanation techniques to explain the working of Ai model.

Data Explanation

Platform facilitates the user to upload data file and get insights from data and also try to find hidden patterns.

What-if tool

What-if tools helps the user to see the model's prediction for unknow or real world instance and explains the predict at the same time.
